# Наместник — Godot 4 (C#)

Цифровая версия: ядро правил, соло, хост/клиент.

## Требования

- **Godot 4.3+** (.NET / mono)
- **.NET SDK 9+** для тестов (`dotnet test`); игра собирается как `net8.0` через Godot.NET.Sdk

## Запуск

1. Откройте `game/` в Godot (.NET).
2. F5 — главное меню.

| Режим | Поведение |
|---|---|
| **Соло** | Локальный движок + виртуал на аукционе; **Отмена** последнего действия |
| **Создать игру** | Хост на порту `7777`; при первом клиенте — партия на 2 человек |
| **Подключиться** | Клиент шлёт команды хосту, получает снапшот состояния |

## Тесты

```bash
cd game
dotnet test Namestnik.Tests/Namestnik.Tests.csproj
```

Покрывают: загрузку карт, колоды 48+аукцион, консервацию камней, клон/undo, #66 без дубля ParkedGems, сериализацию команд/снапшота.

## Что есть

- Аукцион (sealed bid, атака, пас с выбором камней)
- Развитие (пирамида, L5, законы on-play)
- Финальный подсчёт
- Undo в соло + отмена черновика постройки карты в UI
- Сеть: RPC команд + полный снапшот (2 игрока)
- **Арт карт** (`assets/cards/`) + тултипы стоимости/наград
- Визуальный стол: пирамида, аукцион, рука

## Ограничения сети

- Только 2 игрока при первом join
- Снапшот без тумана войны (ставки видны)
- Без реконнекта / host migration
